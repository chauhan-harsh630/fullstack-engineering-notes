## 🛠 How to Consume These Notes

1. **Follow the Directories Sequentially**: Navigate the folders primarily from `01` upwards. The concepts are meant to build upon one another sequentially.
2. **Review Examples In-Depth**: When exploring JavaScript or React, don't just read the code snippets. Use your terminal/editor to implement the pieces locally.
3. **Keep the Cheatsheets Handy**: Files like `git-commands-cheatsheet.md` or `CSS.md` are excellent reference materials whenever you are working on practical projects.
